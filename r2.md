## How to use this

* **Budget view:** Sum each column across VS2–VS5 for a monthly total FTE curve (peaks around months 10–15, then taper).
* **Hiring view:** Use the first 3 months to lock Leads, SRE/DevOps, and core VS2 engineers; ramp VS3/VS4 from month 4.
* **Orals defense:** “Spine first (VS2+VS5), then VS3+VS4, scale in months 10–15, handover in 16–18.”
